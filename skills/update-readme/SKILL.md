---
name: agent-infra-update-readme
description: Use when the user asks to write or update a README file for a project or sub-package
---

Write or update a README using the README template from claude.md.

Read the template before writing. The template contains two variants: one for project-level READMEs and one for sub-package READMEs. Choose the appropriate variant based on context. If $ARGUMENTS is provided, use it as the target directory. Otherwise ask the user which project or package to write the README for.
